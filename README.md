TripLens – Smart Travel Planning System

TripLens is a full-stack travel planning web application that helps users plan their trips efficiently by selecting routes, modes of transport, and generating itinerary plans based on user inputs.


Overview:-

TripLens allows users to:
- Enter their **source and destination**
- Select **travel dates**
- Choose a **mode of transport** (Car, Train, Flight)
- View **routes on an interactive map**
- Get **distance and travel time**
- Generate a **basic itinerary plan**

This project is developed as part of the **Post Graduate Diploma in Advanced Computing (PG-DAC)** at **C-DAC ACTS, Pune**.

---

Tech Used:-

#Frontend
- React.js
- Tailwind CSS
- Leaflet.js (for maps)

#Backend
- Java
- Spring Boot (REST API)

#Database
- MongoDB Atlas

#External APIs
- OpenRouteService (ORS) – for route and distance calculation

---

**Features**

- Location-based route finding
- Multiple travel modes support
- Interactive map visualization using Leaflet
- Distance & time calculation
- Date-based travel planning
- Itinerary generation
- Fast and responsive UI
